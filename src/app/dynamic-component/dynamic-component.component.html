Dynamic Components what's That



Why use Dynamic Component in Angular ?
There arises situations where it's necessary to drop specific components into the view, and the current view should not be tied to URLs at this point in such a scenario router won't help. Dynamic Component loads the various components dynamically into the view. This gives the power to handle the logic in one persistent smart component which is bound to a URL.
