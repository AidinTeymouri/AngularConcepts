9. Importing 3rd Party JS libs in Angular without Typings

Without typings code repo Reference
Step One

npm install jssha --save [using jssha for this demo]

Step two
Add the jssha scripts file in .angular-cli.json file


"scripts": [ "../node_modules/jssha/src/sha.js" ]

Step three
Adding a var in component to be used as a global variable


//using external js modules in Angular
declare var jsSHA: any;
shaObj:any;
hash:string;
this.shaObj = new jsSHA("SHA-512", "TEXT");
this.shaObj.update("This is a Test");
this.hash = this.shaObj.getHash("HEX")

This is value of hash below and it works yayy

HEX: {{hash}}

10. Importing 3rd Party JS libs in Angular with Typings

With typings
Step One

npm install lodash --save [using loadash for this demo]

Step two
Add the typings for the library for popular library you will have types defined like we have for lodash


npm install @types/lodash --save

Step three
Import it in your component Directly


import * as _ from 'lodash';

And it works
